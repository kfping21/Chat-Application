# 树洞 (Tree Hole) - 设计说明文档

## ## 配色方案

本设计采用**深空之境 (Deep Space)** 主题，旨在营造安全、静谧且具有私密感的深夜氛围。

- **主色 (Primary):** `#FFB800` (暖金色) —— 象征黑暗中的微光、温暖的回响。
- **辅助色 (Secondary):** `#1B2436` (深灰蓝) —— 用于卡片背景，增加视觉层次。
- **背景色 (Background):** `#0B101B` (极夜蓝) —— 沉浸式深夜背景。
- **强调色 (Accent):** `#3B82F6` (柔和蓝) / `#EF4444` (提示红)。

## ## 字体

- **标题 (Headings):** `PingFang SC / Semibold` —— 保证中文标题在暗色背景下的清晰度与力量感。
- **正文 (Body):** `PingFang SC / Regular` —— 14px-16px，行高 1.6，确保长文字阅读的舒适度。

## ## 核心页面

1. **首页 (Home Feed):** 采用流式卡片设计，展示用户匿名发布的秘密。包含情绪标签（如：孤独、感动）和基础互动数据。
2. **发现页 (Explore):** 视觉化星空设计。用户通过点击悬浮的“光点”随机开启他人的秘密，增加探索的趣味性。
3. **详情页 (Post Detail):** 沉浸式阅读空间。突出文字内容，下方承载“回响（评论区）”，支持用户进行情感共鸣。
4. **发布页 (Post Editor):** 极简输入界面。提供“心情种子”选择器，引导用户将情绪可视化并进行匿名发布。
5. **消息中心 (Messages/Echoes):** 分为“评论”与“私聊”双频道。通过红点提醒新互动，保护隐私的同时促进深度连接。
6. **个人中心 (Profile):** 展示“我埋下的秘密”与“我收集的回响”，记录用户在树洞里的情感足迹。

## ## 交互说明

- **进入详情:** 点击 **Home Feed** 或 **Explore** 中的卡片/光点，页面通过 `Smart Animate` 放大跳转至 **Post Detail**。
- **发布秘密:** 点击底部导航栏中央的 **“+” (发布)** 按钮，弹出半屏或全屏的 **Post Editor**。
- **私密聊天:** 在 **Post Detail** 底部点击发送图标，或在 **Messages** 列表点击项，跳转至 **Chat** 界面。
- **切换频道:** 在 **Messages** 页面顶部点击“评论/私聊”页签，实现横向滑动切换（Slide Swap）。
- **互动反馈:** 点击“点赞/共鸣”图标，图标由空心变为实心并伴随微光扩散动画。
