## ## 遇到的问题和解决

1. **问题：** 在暗色背景下，多层级卡片堆叠容易导致视觉疲劳，且界面显得沉闷。                                                                                         **解决：** 引入了 **Glassmorphism (玻璃拟态)** 效果和微弱的 **1px 描边 (Stroke)**，并配合微量发光效果，拉开了空间感，增加了灵动性。
2. **问题：** 匿名社交软件如何平衡“陌生感”与“温暖感”？                                                                                                                                      **解决：** 在配色中加入了暖色调的强调色（金色），并使用“种子”、“光点”、“回响”等意象化词汇代替冰冷的技术术语，从视觉和文案上提升产品温度。
